# My Asset Bundle

모든 Asset Store 에셋을 하나로 묶은 Unity 패키지입니다.

## 포함된 에셋

- **ConsolePro** (3.980): Editor Console Pro - 강화된 Unity Console

## 설치 방법

### Git URL로 설치

`Packages/manifest.json`에 추가:

```json
{
  "dependencies": {
    "com.yourname.asset-bundle": "https://github.com/yourname/asset-bundle.git#v1.0.0"
  }
}
```

### 로컬 경로로 설치 (개발용)

```json
{
  "dependencies": {
    "com.yourname.asset-bundle": "file:../asset-bundle"
  }
}
```
